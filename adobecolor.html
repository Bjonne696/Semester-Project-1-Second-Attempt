Adobe color is a great tool!

It features many paletes to select from and you can save these to a library, wich you can conect to a Adobe account to be used with other projects later.

It features a color wheel with ten different modes displaying up to five colors in a palete.

It also has a tool to wich you can upload your own images and display the color code for those images!
